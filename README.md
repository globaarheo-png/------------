# Что готовить?

Telegram-бот на Python и aiogram 3, который помогает быстро выбрать семейное блюдо из продуктов дома.

## Что умеет MVP

- принимает свободный текст с продуктами;
- просит указать продукты, если запрос слишком общий;
- генерирует 3 коротких варианта блюд через GigaChat;
- показывает inline-кнопки `1`, `2`, `3` и `Реши за меня`;
- после выбора показывает подробный рецепт;
- сохраняет рецепт в избранное;
- показывает историю запросов и избранное;
- учитывает семейные настройки: количество людей, возраст ребенка, аллергии и исключенные продукты;
- сохраняет пользователей, настройки, запросы и избранное в Supabase.

## Запуск

1. Создайте и активируйте виртуальное окружение:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Установите зависимости:

```powershell
pip install -r requirements.txt
```

3. Скопируйте `.env.example` в `.env` и заполните значения:

```powershell
Copy-Item .env.example .env
```

Обязательные переменные:

```text
TELEGRAM_BOT_TOKEN=...
GIGACHAT_AUTH_KEY=...
```

Supabase можно не подключать для первого локального запуска. Если `SUPABASE_URL` и `SUPABASE_KEY` пустые, бот будет работать без постоянного хранения.

4. Если нужен Supabase, создайте таблицы из файла `supabase/schema.sql`.

5. Запустите бота:

```powershell
python -m app.main
```

## Команды

- `/start` - запуск и краткая инструкция
- `/help` - помощь и примеры
- `/new` - новый запрос
- `/reset` - то же, что `/new`
- `/again` - еще 3 варианта по последнему запросу
- `/history` - последние запросы
- `/favorites` - избранные рецепты
- `/settings` - настройка семьи одной строкой

Пример для `/settings`:

```text
3 человека; ребенок 2 года; аллергии: орехи, мед; исключить: грибы
```

## Важно

Оценка "подходит ребенку" является бытовой подсказкой, а не медицинской рекомендацией. При аллергиях и медицинских ограничениях ориентируйтесь на рекомендации специалиста.
